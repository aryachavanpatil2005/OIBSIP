# ATM Interface

## Description
This is a Java console-based ATM Interface project developed for the Oasis Infobyte Java Development Internship.

## Features
- Check Balance
- Deposit Money
- Withdraw Money
- View Transaction History
- Exit

## Technologies Used
- Java
- VS Code
