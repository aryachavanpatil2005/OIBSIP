# Number Guessing Game

## Description
This is a Java console-based Number Guessing Game developed for the Oasis Infobyte Java Development Internship.

## Features
- Random number generation (1-100)
- Maximum 7 attempts
- Too High / Too Low hints
- Play Again option

## Technologies Used
- Java
- VS Code
